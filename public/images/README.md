# Bilder für die Website

## SEO Top-3-Seite (Vorher/Nachher-Karten)

Die Vorher/Nachher-Sektion auf `/seo-top-3-in-google` erwartet zwei PNG-Grafiken:

- **Vorher (nicht Top 3):** `seo-vorher-karte.png` – Karte mit roten Markierungen „nicht in Top 3“
- **Nachher (Top 3):** `seo-nachher-karte.png` – Karte mit grünen/gelben Markierungen „Top 3“

Laut Plan stammen die Original-Assets z. B. von:
- `Not_Top_3_in_Google-*.png` → hier als `seo-vorher-karte.png` ablegen
- `Top_3_in_Google-*.png` → hier als `seo-nachher-karte.png` ablegen

Die Grafiken werden mit `object-contain` und ohne Farbfilter dargestellt.
